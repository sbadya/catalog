Подразделения Вооруженных Сил

Описание:

Данное приложение позволяет создавать, изменять, удалять подразделения, а также работать со справочниками.

Список реквизитов:

  1. Наименование
  2. Сведения
  3. Адрес
  4. Телефон
  5. Количество сотрудников
  6. Дополнительная информация
  7. Тип организации
  8. Управляющий
  9. Дата создания

Реализация

Для разработки приложения использовался стек MEAN

  1. Серверная часть - Node.js, Express
  2. СУБД - MongoDB
  3. Клиентская часть - AngularJS, HTML, CSS
  4. Система шаблонизации на стороне сервера - Jade
